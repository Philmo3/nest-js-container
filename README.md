## Description

Small template using nestjs with authentification.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ docker compose up --build

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
